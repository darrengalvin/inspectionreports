Format Summary

Quantitative

  • • Slider (1-10)
  • • Star Rating
  • • Number Input
  • • Multi-dimension

Single Choice

  • • Dropdown
  • • Radio Buttons
  • • Toggle (Yes/No)
  • • Segmented Control
  • • Likert Scale

Free Form

  • • Short Text
  • • Long Text
  • • Quote Capture
  • • Multi-select

Each question in the inspection can use any of these formats.
The format is configured per-question in the section data.
